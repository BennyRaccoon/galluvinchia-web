# Galluvinchia Wiki

> *"In Galluvinchia, the gods do not dwell in distant heavens — they walk the same dust as mortals, bleed the same blood, and hunger for the same things."*

Welcome to the official compendium of **Fables of Galluvinchia**, a tabletop roleplaying game set in a world where divinity is not a distant ideal but a living, breathing, and sometimes terrifying presence.

This wiki is a living document. Pages grow as legends are made.
